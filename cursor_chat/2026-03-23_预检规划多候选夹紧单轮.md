# 选项 2：预检/规划多候选，夹紧失败不重试

## 行为

- **`generateCableSideGrasps`** 重新对外暴露，返回按 score 排序的候选列表。
- **`handlePick`**：对候选循环；**预检失败 / INIT / PLAN** → `continue` 试下一候选。
- **执行成功** → `return true`。
- **`failed_no_grip`（夹紧超时）** → `retreatToReady()`，置 IDLE + `last_error_`，**不再**试下一候选。
- **其它执行失败** → `continue` 试下一候选。
- **全部失败** → `CABLE_SIDE_GRASP: 所有侧抓候选均失败`。
