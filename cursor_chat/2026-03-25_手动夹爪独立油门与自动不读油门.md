# 2026-03-25 手动夹爪油门与自动夹爪不读油门

## 行为

- **手动**：6 臂用 `scale_arm = _speed_scale_for(left, speed_axis_index)`；夹爪用独立 `scale_grip`：`gripper_speed_joy`（left/right）+ `gripper_speed_axis_index`（-1 则与 `speed_axis_index` 相同）。`manual_gripper_use_throttle: false` 时夹爪比例固定 1.0。
- **自动**：`open_gripper` / `close_gripper` 仅键位上升沿，代码注释标明与油门无关（本身未读 scale）。

## 参数

- `manual_gripper_use_throttle`, `gripper_speed_joy`, `gripper_speed_axis_index`
- 调试日志 `log_joint_targets_debug` 打印 `臂油门` 与 `夹爪油门`。

## 实现

- `_speed_scale` 改为 `_speed_scale_for(joy, axis_index)`。
