# 2026-03-25 手柄桥接参数统一为度（°）

## 变更

- `joy_manipulator.yaml` / `joy_manipulator_node`：**限位、夹爪开/合目标角、臂/夹爪手动角速度** 均为 **度**（`*_deg` / `*_deg_s`）。
- 已移除 `manual_*_rad_s`、`arm_joint_*_limits_rad`、`hand_q*_open_rad` 等弧度参数字段名。
- **joint_states 与 FollowJointTrajectory 仍为弧度**（ROS 惯例）；节点在初始化与积分链路内乘以 `π/180` 换算。

## 默认

- 臂限位 ±180°；夹爪开约 ±22.92°（原 0.4 rad）；满油门 60°/s（臂）、25°/s（夹爪）。
